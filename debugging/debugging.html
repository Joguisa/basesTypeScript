¿Qué es el archivo tsconfig y para qué sirve?

tsconfig.json es un archivo de configuración en TypeScript que se utiliza para especificar opciones y configuraciones para el compilador de TypeScript (tsc). Este archivo permite definir cómo se deben compilar los archivos TypeScript en un proyecto.

Incluir y Excluir Carpetas y/o Archivos

En el tsconfig se agrega

"carpeta/archivo.extension"
]
Para incluir:
En el tsconfig se agrega
"include": [
"carpeta/archivo.extension"
]

OutFile

outFile es una opción de configuración en TypeScript que te permite especificar un único archivo de salida en lugar de generar múltiples archivos JavaScript separados después de la compilación. Es decir, puedes generar un único archivo JavaScript concatenando todos los archivos TypeScript en uno solo.
